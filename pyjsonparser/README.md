# pyjsonparser

A Python package for parsing and working with JSON data.

## Usage

```python
import pyjsonparser

# Parse JSON string
data = pyjsonparser.parse('{"name": "John", "age": 30}')
print(data)
```

## Features

- Simple and intuitive JSON parsing
- Lightweight and fast
- Error handling for malformed JSON

## Requirements

- Python 3.7+
